PepsiAPI

Follow the steps below to use the PepsiAPI in your Visual Studio project.

Step 1: Download the Pepsi-API

Click the button below to download the ExecutorAPI.dll file.

Download Pepsi-API

Step 2: Use the Attach Function

Here’s how to attach the API to the Roblox process:

ExecutorAPI.Instance.Attach();

Step 3: Use the QuickScript Function

Here’s how to use the QuickScript Function:

ExecutorAPI.QuickScript("Luau", "print('Hello from Luau!')");

Step 4: Execution Function

To execute a Lua script in Roblox, use the following command:

ExecutorAPI.Execute(textBox1.Text);
